pydockstats.py is a script that generates the Predictiveness and ROC curves given a dataset of decoys and ligands with their respectives IDs or names, docking scores and activities (0 or 1). The input data (.csv) must be separated into columns for each docking program data:

Example:

id_program1 | scores_program1 | activity_program1 | id_program2 | scores_program2 | activity_program2


The code runs at the command line:

python pydockstats.py -f <csv input file> -p <name of the Docking programs described in your CSV file> -o  <image name for the output file (ex: "out.png")>

For example:

> python pydockstats.py -f input_data.csv -p gold,vina,dockthor -o plt_out.png
